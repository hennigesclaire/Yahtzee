# Yahtzee
## General Introduction

- This is a Yahtzee game, based on the titular dice game, written in Java.

## Features

- Dice Rolling and Selection
- Turn Manager
- Score Validation and Calculation
- New Special Conditions (Evens and Odds)
- Easy, Medium, and Hard A.I. Players

## Technologies

- Java Development Kit 18
- Apache NetBeans IDE 23
- Java Swing Library
- GitHub and GitHub Desktop (for Repository Hosting)
- FontStruct (for Adding Dice Pips to Dice)

## How To Run

- Download the Code
- Open the Project in Apache NetBeans
- Run the Project
